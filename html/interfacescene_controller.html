- (void) addObjectToScene: (MCSceneObject*)  sceneObject

将需要加载的场景对象加载到“预备加载”队列中。并且唤醒场景对象。

we don't actualy add the object directly to the scene. this can get called anytime during the game loop, so we want to queue up any objects that need adding and add them at the start of the next game loop

实际的加入场景对象将会推迟到新的游戏循环的开始阶段。

- (void) removeObjectFromScene: (MCSceneObject*)  sceneObject

将需要加载的场景对象移除到“预备移除”队列中。

similar to adding objects, we cannot just remove objects from the scene at any time. we want to queue them for removal and purge them at the end of the game loop

实际的移除场景对象将推迟到当前游戏循环的结束
