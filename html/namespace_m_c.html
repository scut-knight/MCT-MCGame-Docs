详细描述

命名空间MC中定义了与各种状态有关的常量和枚举类型 注意这里面的东西将会被频繁使用到，尤其在DataModel部分 还有，枚举类里枚举变量的顺序都是有特定意义的

类型定义说明

typedef enum MC::_ActionType MC::ActionType

the rull(rule?) action type

typedef enum MC::_AppliedRuleType MC::AppliedRuleType

There is two types of rules to be applied.

typedef enum MC::_AxisType MC::AxisType

3D coordinate axis

typedef enum MC::_ColorCombinationType MC::ColorCombinationType

表示立方块带有的颜色组合(色块组合) 注意以下的枚举类型的顺序是固定的，分别表示[0][0][0] 到 [2][2][2]的三维坐标上立方块的位置 name after direction in this order:front(F), back(B), left(L), right(R), up(U), down(D) e.g. BLD is the cube that has three colors:BACK_COLOR, LEFT_COLOR, DOWN_COLOR

typedef enum MC::_CubieType MC::CubieType

three types of cubies

typedef enum MC::_ExpType MC::ExpType

the expression type

typedef enum MC::_FaceColorType MC::FaceColorType

define the face color type to apart color data from model

typedef enum MC::_FaceOrientationType MC::FaceOrientationType

define the orientation of face

typedef enum MC::_InformationType MC::InformationType

the getting information type

typedef enum MC::_LayerRotationDirectionType MC::LayerRotationDirectionType

define the rotation direction, clockwise and anticlockwise

typedef enum MC::_NodeType MC::NodeType

the node pattern type

typedef enum MC::_PatternType MC::PatternTyp

the pattern type

typedef enum MC::_RotationLayerMode MC::RotationLayerMode

There is three types of rotation relevant to axis, layer and direction.

typedef enum MC::_RotationResult MC::RotationResult

the result indicates whether the current rotation accords the required rotation 反应当前旋转的结果，尤其是是否与所需的旋转相符合
